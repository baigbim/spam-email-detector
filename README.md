# Spam Email Detector 📧🚫

## Project Overview
A Machine Learning model that detects whether 
an email is Spam or Not Spam (Ham) with 99% accuracy!
Just like Gmail's spam filter!

## Results 🎯
| Metric | Score |
|--------|-------|
| Accuracy | 99% |
| Model | Naive Bayes |
| Total Emails | 5572 |
| Spam Emails | 747 |
| Ham Emails | 4825 |

## How it Works 🔧
Step 1: Email text lo
Step 2: CountVectorizer se numbers mein badlo
Step 3: Naive Bayes model classify kare
Step 4: Spam ❌ ya Ham ✅ batao!

## Sample Predictions 🎯
❌ SPAM: "Free money! Click here now! Win prize!"
✅ HAM : "Hey, are you coming to office tomorrow?"
❌ SPAM: "Congratulations! You won $1000! Claim now!"
✅ HAM : "Can we meet for lunch today?"
